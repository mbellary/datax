# Phase 2 Planning Artifacts

This folder contains Phase 2 planning artifacts for the Datax fork-first
migration and adapter-first product architecture.

Read these files in order:

1. `platform.md` describes the product direction and platform boundary.
2. `Recommended-Datax-Phase2-Execution-Model.md` defines how each Phase 2
   implementation phase must be planned, tracked, validated, and closed.
3. `Provisional-Datax-Migration-Plan-Phase2.md` defines the implementation-ready
   Phase 2 roadmap.

The older repository-level `docs/plans/Provision-Datax-Migration-Plan-Phase2.md`
is retained as historical context, but it assumes a greenfield repo and is not
the active Phase 2 plan.
